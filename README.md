# University of Hradec Králové Java – UPROM / PRO1 (Task Collection)

A personal collection of Java programming tasks and solutions created during UHK courses **UPROM** and **PRO1**.

> Goal: keep all assignments in one place, with clean structure, runnable code, and short notes for each task.

---

## Contents

- **UPROM** – introductory programming tasks (Java basics, control flow, arrays, methods, OOP intro)
- **PRO1** – follow-up tasks (OOP, collections, exceptions, files, simple algorithms)

Repository includes:
- solved assignments (and sometimes multiple approaches)
- helper utilities / reusable snippets
- notes or comments when something is tricky
